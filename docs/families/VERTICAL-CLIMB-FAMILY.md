# Vertical Climb Family

## Purpose

The vertical climb family supports endless or authored 2.5D climb games with ledge routes, swing movement, ascent chunks, cloud zones, click/hover input, climb cameras, diegetic feedback, risk evaluation, and high-level click-to-climb coordination.

## Known kits

- `protokit-core`
- `content-palette-kit`
- `layered-object-kit`
- `vertical-climb-core`
- `ledge-route-kit`
- `simple-swing-kit`
- `endless-ascent-kit`
- `cloud-zone-kit`
- `climb-input-kit`
- `climb-camera-kit`
- `diegetic-feedback-kit`
- `climb-risk-kit`
- `next-ledge-kit`

## Recommended composition

```txt
content palette
layered objects
vertical climb core
ledge route
```
