# cozy-hero-cloud-form-kit

Renderer-agnostic kit for one simple readable cloud form above Cozy Island.

## Purpose

Get the cloud silhouette right before adding layers, weather, LOD, or storm behavior.

## Owns

```txt
single hero puff cloud descriptor
compact lobe silhouette
above-island placement
camera-corridor avoidance intent
cached point-cloud render hint
soft lighting hints
```

## Does not own

```txt
Three.js renderer objects
GPU buffers
particle shaders
DOM
```

## Fits with

```txt
mattatz-cloud-core-kit
mattatz-cloud-layer-kit
mattatz-cloud-lighting-kit
mattatz-cloud-lod-kit
```
