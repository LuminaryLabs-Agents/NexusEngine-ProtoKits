# Procedural Cloud Layer Kit

Renderer-agnostic procedural cloud layer descriptors.

## Services

- `engine.proceduralCloudLayer.generateAround(position)`
- `engine.proceduralCloudLayer.getDescriptors(context)`
- `engine.proceduralCloudLayer.snapshot()`

## Provides

- `sky:cloud-layers`
- `cloud:layer-descriptors`
- `render:cloud-descriptors`

## Boundary

This kit does not create cloud meshes, shaders, textures, or renderer state.
