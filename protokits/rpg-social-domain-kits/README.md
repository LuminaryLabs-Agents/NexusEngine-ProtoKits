# rpg-social-domain-kits

## Family boundary

This README documents the RPG social domain kit family boundary. Individual kit manifests and READMEs will follow.

## Purpose

RPG social domain kits provide reusable services for dialogue lines, relationship state, NPC schedules, shop inventories, and quest threads.

## Ownership

The kits should own reusable social/domain rules and state surfaces. Specific characters, lines, schedules, shops, and quests should remain data.
