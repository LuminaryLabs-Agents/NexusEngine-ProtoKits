# Mattatz Cloud Weather Kit

Preset composition kit for the `mattatz-clouds-domain`.

## Owns

- named cloud weather presets
- cloud count defaults
- coverage defaults
- density defaults
- softness defaults
- wind speed defaults
- layer distribution weights
- storm-front toggles for cumulonimbus descriptors

## Presets

- `clear`
- `scattered`
- `overcast`
- `storm-front`
- `mountain-fog`
- `sunrise-haze`
- `high-windy-sky`

## Public implementation

```js
import { createMattatzCloudWeatherKit } from "@luminarylabs/nexusengine-protokits/mattatz-clouds-domain";
```
