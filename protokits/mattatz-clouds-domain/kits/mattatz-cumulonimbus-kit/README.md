# Mattatz Cumulonimbus Kit

Vertical storm-cloud descriptor kit for the `mattatz-clouds-domain`.

## Owns

- tall cloud tower descriptors
- core scale
- base and top altitude
- anvil cap strength
- rain shaft descriptor strength
- underside darkness

## Why separate

Cumulonimbus clouds need vertical structure and stacked volume intent. Keeping this separate lets fair-weather layers stay simple.

## Public implementation

```js
import { createMattatzCumulonimbusKit } from "@luminarylabs/nexusengine-protokits/mattatz-clouds-domain";
```
