# environment-kits

## Family boundary

This README documents the environment kit family boundary. Individual kit manifests and READMEs will follow.

## Purpose

Environment kits support reusable terrain, atmosphere, weather, vegetation, environmental feedback, and renderer-agnostic scene fidelity services.

## Ownership

Environment kits may own deterministic environment state and descriptors. Specific levels, art direction presets, assets, and renderer object pooling should remain in deploy data, presets, asset packs, hosts, or renderer adapters.
