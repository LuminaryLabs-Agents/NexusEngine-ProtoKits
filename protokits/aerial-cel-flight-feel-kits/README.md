# aerial-cel-flight-feel-kits

## Family boundary

This README documents the aerial cel flight feel family boundary. Individual kit manifests and READMEs will follow.

## Purpose

Aerial cel flight feel kits support stylized flight control, visual readability, camera-friendly movement feel, and renderer-agnostic flight presentation descriptors.

## Ownership

These kits should own reusable flight-feel state and descriptors. Specific aircraft, dragon, bird, or game-branded content should remain data, presets, sequences, deploy manifests, or experiments.
