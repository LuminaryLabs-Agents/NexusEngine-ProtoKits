# scoped-rpg-domain-kits-batch-02

## Family boundary

This README documents the scoped RPG domain kit family boundary. Individual kit manifests and READMEs will follow.

## Purpose

Scoped RPG domain kits provide reusable combat, enemy, agent, guard, parry, mana, status, vegetation, route, ground-contact, world-zone, and interaction domains.

## Ownership

The kits should remain small and reusable. Specific enemies, levels, RPG content, tuning tables, and branded encounters should remain data, presets, scenes, or deploy manifests.
