# aerial-biome-fidelity-kits

## Family boundary

This README documents the aerial biome fidelity family boundary. Individual kit manifests and READMEs will follow.

## Purpose

Aerial biome fidelity kits support large-scale readable biome presentation, environmental material variation, vegetation/prop fidelity, and renderer-agnostic fidelity descriptors for aerial scenes.

## Ownership

Reusable biome/fidelity rules belong here. Game-specific scenery, branded sky worlds, and authored mission content should stay in data, presets, deploy manifests, or experiments.
